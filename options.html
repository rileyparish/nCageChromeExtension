Welcome to nCage+!

This is the options page, where you can specify how you would like the extension to behave.

You can go ahead and close this tab after you've set everything up. From here on out, the script will run on every webpage until the extension is disabled! You can always navigate back to this page from the extension settings.

Image Replacement Configuration

(I recommend somewhere around 1% for moderate subtlety. Seriously!)

This option will allow you to create a selection of images of your choosing, but it will take some preparation.
All you'll need is a list of publicly accessible URLs of the images you want to use (simpler URLs are more likely to work).
These can be images that you uploaded to the internet yourself, or images that others have uploaded.
Paste them below in a comma-separated list. With great power comes great responsibility!
